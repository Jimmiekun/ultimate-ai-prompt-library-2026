# Prompt Template

# The Ultimate AI Prompt Library (2026 Edition)

---

## Purpose

This template defines the standard format for every prompt in the ebook.

Every prompt must follow this structure exactly to maintain consistency and provide maximum value to readers.

---

# Standard Prompt Layout

## Prompt #[Number]

### Title

A short, descriptive title explaining what the prompt does.

Example:

Write a Professional Business Plan

---

### Category

Example:

Business

---

### Difficulty

Choose one:

⭐ Beginner

⭐⭐ Intermediate

⭐⭐⭐ Advanced

---

### Purpose

Briefly explain what the prompt helps the user accomplish.

Maximum:
2–4 sentences.

---

### Copy & Paste Prompt

Place the complete prompt inside a code block.

Example:

```text
Act as an experienced business consultant.

Help me create a complete business plan for my company.

Business Name:
[Business Name]

Industry:
[Industry]

Target Audience:
[Target Audience]

Generate:

• Executive Summary

• SWOT Analysis

• Revenue Model

• Marketing Strategy

• Financial Projections

Use professional language and present the result using clear headings.
```

---

### Customization

Explain what readers should replace.

Example:

Replace:

- [Business Name]
- [Industry]
- [Target Audience]

with your own information.

---

### Expected Output

Readers should receive something like:

- Executive Summary
- SWOT Analysis
- Marketing Plan
- Financial Forecast
- Growth Strategy

---

### Best AI Models

This prompt works well with:

- ChatGPT
- Claude
- Gemini

Optional:

- Microsoft Copilot
- Perplexity
- NotebookLM

---

### Pro Tip

Provide one practical tip that helps users get even better results.

Example:

"After receiving the first response, ask the AI to improve the financial projections based on a startup budget."

# Prompt Quality Standards

✔ Solve one real-world problem.

✔ Be easy to copy and paste.

✔ Be easy to customize.

✔ Work across multiple AI platforms.

✔ Deliver immediate value.
